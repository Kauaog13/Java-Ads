# Projeto Quiz POO II

Este é um projeto de Quiz desenvolvido em Java utilizando a biblioteca Swing para a disciplina de Programação Orientada a Objetos II. O objetivo do sistema é apresentar uma série de perguntas de múltipla escolha e contabilizar a pontuação final do usuário.

## 🚀 Estrutura do Projeto
O projeto está organizado da seguinte forma:
* **src/**: Contém o código-fonte Java (`.java`).
* **bin/**: Contém os arquivos compilados (`.class`).
* **Quiz.java**: Classe principal que gerencia o JFrame e a navegação entre as telas.
* **PanePerguntaX.java**: Classes que representam cada uma das 5 perguntas do quiz.

## 🛠️ Tecnologias Utilizadas
* **Java**
* **Swing** (para interface gráfica)
* **Visual Studio Code** (Ambiente de desenvolvimento)

## 🎮 Como Funciona
1. Ao iniciar a aplicação através da classe `Quiz.java`, a primeira pergunta é exibida.
2. O usuário seleciona uma opção e clica no botão **Verificar**.
3. O sistema informa se a resposta está correta ou incorreta e atualiza a pontuação global.
4. Após responder a quinta pergunta, o sistema exibe a pontuação final acumulada e encerra a aplicação.

## ⚙️ Como Executar
1. Certifique-se de ter o JDK instalado em sua máquina.
2. Compile os arquivos dentro da pasta `src`.
3. Execute a classe `Quiz.java`.
